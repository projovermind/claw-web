v{{VERSION}} — Claude Agent Management Platform
Claude AI 에이전트를 웹에서 관리하고 실시간으로 대화할 수 있는 플랫폼입니다.
프로젝트별 에이전트 구성, 스킬 시스템, 원격 접속을 지원합니다.

주요 기능

  • 실시간 채팅 — Claude CLI 스트리밍 응답
  • 에이전트 관리 — 모델, 시스템 프롬프트, 도구 권한 설정
  • 프로젝트 관리 — 프로젝트별 에이전트 그룹화
  • 스킬 시스템 — 재사용 가능한 프롬프트 템플릿
  • 원격 접속 — ngrok / Cloudflare Tunnel로 어디서든 접속

지원 백엔드

  • Claude (Anthropic) — API 키 또는 OAuth
  • OpenAI · OpenRouter · DeepSeek
이 설치 프로그램은 다음을 수행합니다:
1. 인증번호 확인 → 2. Node.js 확인 / 자동 설치 → 3. 서버 & 클라이언트 의존성 설치 → 4. 클라이언트 빌드 → 5. Claude CLI 확인 → 6. 자동 시작 등록

설치 중 macOS 알림으로 진행 상황을 안내합니다. (1~3분 소요)
