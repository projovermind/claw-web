서버가 자동으로 시작됩니다. 브라우저에서 위 주소로 접속하세요.

초기 설정

  1. Settings → AI Backends에서 Anthropic API 키 입력
  2. Agents 탭에서 에이전트 추가
  3. Chat 탭에서 대화 시작!

원격 접속 URL 확인

설치 중 터널을 선택하셨다면 URL 이 자동 생성되었습니다.
• 고정 URL: cat ~/.claw-web/.tunnel-url (영구, 재부팅 OK)
• 빠른 URL: grep trycloudflare /tmp/claw-web-tunnel.log (재부팅마다 바뀜)
• ngrok: curl -s localhost:4040/api/tunnels | grep public_url

🔑 Bearer 토큰: cat ~/.claw-web/.bearer-token
외부 브라우저/폰에서 접속 시 Settings → Access 탭에 토큰 입력 필요.

유용한 명령어

서버 로그: tail -f /tmp/claw-web.log
서버 중지: launchctl unload ~/Library/LaunchAgents/com.claw-web.server.plist
서버 시작: launchctl load ~/Library/LaunchAgents/com.claw-web.server.plist
터널 중지: launchctl unload ~/Library/LaunchAgents/com.claw-web.tunnel.plist
터널 재시작: launchctl load ~/Library/LaunchAgents/com.claw-web.tunnel.plist
수동 실행: cd /usr/local/lib/claw-web && npm start

고정 URL 재설정 (나중에 변경 시)

설치 후에도 고정 URL 추가/변경 가능:
1. brew install cloudflared && cloudflared tunnel login
2. cloudflared tunnel create claw-web
3. cloudflared tunnel route dns claw-web claw.내도메인.com
4. ~/.cloudflared/config.yml 작성 후 터널 재시작
