설치 전 확인사항

필수 요구사항

  • Node.js 20+ — nodejs.org에서 설치
  • Claude CLI (선택) — npm install -g @anthropic-ai/claude-code

설치 후 설정

  1. 브라우저에서 http://localhost:3838 접속
  2. Settings → AI Backends에서 API 키 입력
  3. Agents 탭에서 에이전트 생성
  4. Chat 탭에서 대화 시작!
API 키 안내: Anthropic API 키가 필요합니다.
console.anthropic.com에서 발급받으세요.
없어도 claude login으로 OAuth 인증하면 사용 가능합니다.

원격 접속 (선택) — 외부에서 내 Mac 접속

설치 직후엔 localhost:3838만 됩니다. 핸드폰 / 다른 노트북 / 외부에서 접속하려면 터널이 필요합니다. 두 가지 방법:

① ngrok (간단, 5분)

  1. dashboard.ngrok.com/signup 가입 (무료)
  2. 설치: brew install ngrok
  3. 대시보드에서 authtoken 복사 → ngrok config add-authtoken YOUR_TOKEN
  4. 실행: ngrok http 3838
  5. 출력된 https://xxxx.ngrok-free.app 로 어디서든 접속

② Cloudflare Tunnel (무료, 즉시 사용)

  1. 설치: brew install cloudflared
  2. 실행: cloudflared tunnel --url http://localhost:3838
  3. 터미널에 표시되는 trycloudflare.com URL 로 접속 (가입 불필요)

원격 접속 시 반드시 Settings → Access 탭에서 Bearer 토큰을 설정해 인증을 활성화하세요. 안 그러면 누구나 접속 가능.

설치 위치: /usr/local/lib/claw-web
데이터 위치: ~/.claw-web/
서버 포트: 3838
