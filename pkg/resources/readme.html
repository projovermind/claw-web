설치 전 확인사항

필수 요구사항

  • Node.js 20+ — nodejs.org에서 설치
  • Claude CLI (선택) — npm install -g @anthropic-ai/claude-code

설치 후 설정

  1. 브라우저에서 http://localhost:3838 접속
  2. Settings → AI Backends에서 API 키 입력
  3. Agents 탭에서 에이전트 생성
  4. Chat 탭에서 대화 시작!
API 키 안내: Anthropic API 키가 필요합니다.
console.anthropic.com에서 발급받으세요.
없어도 claude login으로 OAuth 인증하면 사용 가능합니다.

원격 접속 — 3가지 방법 중 선택

설치 도중 다이얼로그에서 자동 설정됩니다. 아래는 각 옵션의 설명:

🆓 ① 빠른 URL (Cloudflare Quick Tunnel)

  • 무료, 가입 불필요, 대역폭 무제한
  • 단점: 재부팅할 때마다 URL 바뀜
  • 간헐적 사용에 적합

🆓 ② ngrok (월 1GB 제한)

  • dashboard.ngrok.com/signup 가입 필요 (무료)
  • 설치 중 authtoken 붙여넣으면 자동 구성
  • ⚠️ 월 1GB 한도 — Claude 상시 사용 시 3~7일에 소진 (ERR_NGROK_725)

⭐ ③ 고정 URL (Cloudflare Named Tunnel) — 추천

  • 영구 URL: claw.내도메인.com 재부팅/네트워크변경에도 불변
  • 대역폭 무제한
  • 준비물: 도메인 1개 (연 $3~13 수준)
    • 가장 쉬움: Cloudflare Registrar 원가판매
    • 가장 쌈: Porkbun (.cc 첫해 $3.40)
  • 자동화: 설치 도중 다이얼로그에 호스트명(예: claw.mydomain.com) 입력 → 브라우저에서 Cloudflare '권한 부여' 클릭 1번 → 나머지 모두 자동

⚠️ 원격 접속 시 필수: 설치 관리자가 Bearer 토큰을 자동 생성해 web-config.json 에 저장합니다. 설치 완료 다이얼로그에서 [토큰 복사] 버튼으로 클립보드에 담기 → 폰/외부 브라우저에서 Settings → Access 탭에 입력.

설치 위치: /usr/local/lib/claw-web
데이터 위치: ~/.claw-web/
서버 포트: 3838
